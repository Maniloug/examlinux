- La différence entre GNU et Linux 

     GNU : 
      C'est un système d'exploitation libre de type Unix. 
      Il a été initié par Richard Stallman dans les années 1980. 
      Il est composé d'un ensemble de logiciels libres (outils, bibliothèques, applications). 
      Il est le projet qui a donné naissance à la licence GPL. 
      Linux : 
      C'est un noyau de système d'exploitation libre, créé par Linus Torvalds en 1991. 
      Il sert de base pour de nombreux systèmes d'exploitation. 
      Il gère les ressources matérielles de l'ordinateur. 
      En résumé : 
      GNU fournit les outils et les applications, tandis que Linux fournit le noyau. 
      Ensemble, ils forment ce que l'on appelle souvent « GNU/Linux » ou « Linux », un système d'exploitation complet.



- L'inventeur de Linux est Linus Torvalds, un informaticien finlandais. Il a créé le noyau Linux en 1991. Linus Torvalds est également à l'origine de Git. Git est un système de gestion de versions distribué, largement utilisé dans le développement de logiciels.
